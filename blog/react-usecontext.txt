f:T18c8,<p><strong>useContext</strong> adalah hook di React yang memungkinkan Anda <strong>mengakses nilai context</strong> secara langsung di dalam komponen tanpa harus melewati props secara manual (prop drilling). Berikut penjelasan lengkap beserta contoh penggunaannya:</p>
<hr>
<h3><strong>1. Konsep Dasar Context</strong></h3>
<p>Context digunakan untuk <strong>berbagi data</strong> antar komponen dalam hierarki komponen tanpa harus mengirim props secara eksplisit. Contoh kasus: tema aplikasi, data user yang login, atau preferensi bahasa.</p>
<hr>
<h3><strong>2. Langkah Menggunakan useContext</strong></h3>
<h4>a. <strong>Buat Context</strong></h4>
<p>Gunakan createContext() untuk membuat context:</p>
<pre><code class="language-jsx">import { createContext } from 'react';

// 1. Buat context dengan nilai default
const ThemeContext = createContext('light'); // 'light' adalah nilai default
</code></pre>
<h4>b. <strong>Provider Component</strong></h4>
<p>Bungkus komponen anak dengan Provider untuk menyediakan nilai context:</p>
<pre><code class="language-jsx">function App() {
  const [theme, setTheme] = useState('dark');

  // 2. Berikan nilai context ke komponen anak
  return (
    &#x3C;ThemeContext.Provider value={theme}>
      &#x3C;Navbar />
      &#x3C;button onClick={() => setTheme('dark')}>Ganti Tema&#x3C;/button>
    &#x3C;/ThemeContext.Provider>
  );
}
</code></pre>
<h4>c. <strong>Konsumsi Context dengan useContext</strong></h4>
<p>Akses nilai context di komponen anak:</p>
<pre><code class="language-jsx">import { useContext } from 'react';

function Navbar() {
  // 3. Ambil nilai context
  const theme = useContext(ThemeContext);

  return (
    &#x3C;nav style={{ background: theme === 'dark' ? '#333' : '#eee' }}>
      Navbar ({theme})
    &#x3C;/nav>
  );
}
</code></pre>
<hr>
<h3><strong>3. Contoh Lengkap: Tema Aplikasi</strong></h3>
<h4>a. Buat Context dan Provider</h4>
<pre><code class="language-jsx">// ThemeContext.js
import { createContext, useState } from 'react';

export const ThemeContext = createContext('light');

export function ThemeProvider({ children }) {
  const [theme, setTheme] = useState('light');

  return (
    &#x3C;ThemeContext.Provider value={{ theme, setTheme }}>
      {children}
    &#x3C;/ThemeContext.Provider>
  );
}
</code></pre>
<h4>b. Gunakan Provider di Root Komponen</h4>
<pre><code class="language-jsx">// App.js
import { ThemeProvider } from './ThemeContext';

function App() {
  return (
    &#x3C;ThemeProvider>
      &#x3C;Header />
      &#x3C;Content />
    &#x3C;/ThemeProvider>
  );
}
</code></pre>
<h4>c. Akses Context di Komponen Anak</h4>
<pre><code class="language-jsx">// Header.js
import { useContext } from 'react';
import { ThemeContext } from './ThemeContext';

function Header() {
  const { theme, setTheme } = useContext(ThemeContext);

  return (
    &#x3C;header style={{ background: theme === 'dark' ? '#333' : '#fff' }}>
      &#x3C;button onClick={() => setTheme(theme === 'dark' ? 'light' : 'dark')}>
        Toggle Tema
      &#x3C;/button>
    &#x3C;/header>
  );
}
</code></pre>
<hr>
<h3><strong>4. Aturan Penting</strong></h3>
<ol>
<li>
<p><strong>Provider Harus Membungkus Komponen Anak</strong><br>
Pastikan komponen yang mengakses context berada di dalam &#x3C;Context.Provider>.</p>
</li>
<li>
<p><strong>Nilai Context Bisa Berupa Apa Saja</strong><br>
Nilai context bisa berupa string, objek, array, atau bahkan fungsi:</p>
<pre><code class="language-jsx">&#x3C;UserContext.Provider value={{ user, setUser, logout }}>
  {children}
&#x3C;/UserContext.Provider>
</code></pre>
</li>
<li>
<p><strong>Gunakan Context untuk Data yang Sering Diakses Global</strong><br>
Contoh: autentikasi, tema, bahasa, atau konfigurasi.</p>
</li>
</ol>
<hr>
<h3><strong>5. Perbandingan useContext vs. Props Drilling</strong></h3>
<div class="overflow-x-auto bg-lime-100"><table>
<thead>
<tr>
<th><strong>Kriteria</strong></th>
<th><strong>useContext</strong></th>
<th><strong>Props Drilling</strong></th>
</tr>
</thead>
<tbody>
<tr>
<td><strong>Kompleksitas</strong></td>
<td>Minimal (akses langsung)</td>
<td>Tinggi (kirim props ke banyak level)</td>
</tr>
<tr>
<td><strong>Keterbacaan</strong></td>
<td>Lebih mudah</td>
<td>Rentan "spaghetti code"</td>
</tr>
<tr>
<td><strong>Kasus Penggunaan</strong></td>
<td>Data global (tema, user)</td>
<td>Data lokal antar 1-2 komponen</td>
</tr>
</tbody>
</table></div>
<hr>
<h3><strong>6. Best Practices</strong></h3>
<ul>
<li>
<p><strong>Pisahkan Context Berdasarkan Fungsi</strong><br>
Buat context terpisah untuk tema, user, dll., daripada satu context besar.</p>
<pre><code class="language-jsx">const ThemeContext = createContext();
const UserContext = createContext();
</code></pre>
</li>
<li>
<p><strong>Hindari Re-render Tidak Perlu</strong><br>
Jika nilai context sering berubah, gunakan useMemo atau memo untuk optimisasi:</p>
<pre><code class="language-jsx">const value = useMemo(() => ({ theme, setTheme }), [theme]);
</code></pre>
</li>
<li>
<p><strong>Gunakan Custom Hook untuk Context</strong><br>
Contoh:</p>
<pre><code class="language-jsx">export function useTheme() {
  return useContext(ThemeContext);
}
</code></pre>
</li>
</ul>
<hr>
<h3><strong>Kapan Menggunakan useContext?</strong></h3>
<ul>
<li>Saat perlu <strong>berbagi data</strong> ke banyak komponen (misal: user login, tema).</li>
<li>Untuk menghindari <strong>prop drilling</strong> yang rumit.</li>
<li>Jika data <strong>tidak sering berubah</strong> (untuk data dinamis kompleks, pertimbangkan Redux/Zustand).</li>
</ul>
<hr>
<h3>Contoh Lain: User Authentication</h3>
<pre><code class="language-jsx">// UserContext.js
import { createContext, useState } from 'react';

export const UserContext = createContext(null);

export function UserProvider({ children }) {
  const [user, setUser] = useState(null);

  const login = (userData) => setUser(userData);
  const logout = () => setUser(null);

  return (
    &#x3C;UserContext.Provider value={{ user, login, logout }}>
      {children}
    &#x3C;/UserContext.Provider>
  );
}

// Gunakan di komponen:
function Profile() {
  const { user, logout } = useContext(UserContext);

  return (
    &#x3C;div>
      {user ? (
        &#x3C;button onClick={logout}>Logout {user.name}&#x3C;/button>
      ) : (
        &#x3C;p>Silakan login&#x3C;/p>
      )}
    &#x3C;/div>
  );
}
</code></pre>
