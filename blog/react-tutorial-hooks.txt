f:T1695,<p>Berikut daftar hook React.js beserta penjelasan dan contoh penggunaannya:</p>
<hr>
<h3>1. <strong>useState</strong></h3>
<p><strong>Kegunaan</strong>: Mengelola state dalam komponen fungsi.<br>
<strong>Contoh</strong>:</p>
<pre><code class="language-jsx">const [count, setCount] = useState(0); // Inisialisasi state dengan nilai 0

return (
  &#x3C;button onClick={() => setCount(count + 1)}>
    Klik saya: {count}
  &#x3C;/button>
);
</code></pre>
<hr>
<h3>2. <strong>useEffect</strong></h3>
<p><strong>Kegunaan</strong>: Menangani efek samping (side effects) seperti API calls, subscriptions, atau manipulasi DOM.<br>
<strong>Contoh</strong>:</p>
<pre><code class="language-jsx">useEffect(() => {
  console.log("Komponen dimount atau count berubah");
  return () => {
    console.log("Cleanup sebelum efek berikutnya");
  };
}, [count]); // Efek dijalankan ketika &#x3C;code>count&#x3C;/code> berubah
</code></pre>
<hr>
<h3>3. <strong>useContext</strong></h3>
<p><strong>Kegunaan</strong>: Mengakses nilai dari React Context.<br>
<strong>Contoh</strong>:</p>
<pre><code class="language-jsx">const ThemeContext = React.createContext('light');

function Button() {
  const theme = useContext(ThemeContext); // Ambil nilai tema
  return &#x3C;button className={theme}>Tombol&#x3C;/button>;
}
</code></pre>
<hr>
<h3>4. <strong>useReducer</strong></h3>
<p><strong>Kegunaan</strong>: Mengelola state kompleks dengan pola Redux.<br>
<strong>Contoh</strong>:</p>
<pre><code class="language-jsx">const initialState = { count: 0 };

function reducer(state, action) {
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 };
    default:
      return state;
  }
}

const [state, dispatch] = useReducer(reducer, initialState);
&#x3C;button onClick={() => dispatch({ type: 'increment' })}>
  Hitung: {state.count}
&#x3C;/button>
</code></pre>
<hr>
<h3>5. <strong>useCallback</strong></h3>
<p><strong>Kegunaan</strong>: Memoisasi fungsi untuk menghindari re-render tidak perlu.<br>
<strong>Contoh</strong>:</p>
<pre><code class="language-jsx">const handleClick = useCallback(() => {
  console.log("Tombol diklik");
}, []); // Fungsi tidak dibuat ulang kecuali dependency berubah
</code></pre>
<hr>
<h3>6. <strong>useMemo</strong></h3>
<p><strong>Kegunaan</strong>: Memoisasi nilai hasil komputasi mahal.<br>
<strong>Contoh</strong>:</p>
<pre><code class="language-jsx">const expensiveValue = useMemo(() => {
  return a * b; // Hanya dihitung ulang jika &#x3C;code>a&#x3C;/code> atau &#x3C;code>b&#x3C;/code> berubah
}, [a, b]);
</code></pre>
<hr>
<h3>7. <strong>useRef</strong></h3>
<p><strong>Kegunaan</strong>: Menyimpan referensi DOM atau nilai yang persist antar-render.<br>
<strong>Contoh</strong>:</p>
<pre><code class="language-jsx">const inputRef = useRef(null);

&#x3C;input ref={inputRef} />
&#x3C;button onClick={() => inputRef.current.focus()}>
  Fokus Input
&#x3C;/button>
</code></pre>
<hr>
<h3>8. <strong>useImperativeHandle</strong></h3>
<p><strong>Kegunaan</strong>: Menyesuaikan nilai instance yang diekspos ke komponen parent via ref.<br>
<strong>Contoh</strong>:</p>
<pre><code class="language-jsx">const FancyInput = forwardRef((props, ref) => {
  const inputRef = useRef();
  useImperativeHandle(ref, () => ({
    focus: () => inputRef.current.focus()
  }));
  return &#x3C;input ref={inputRef} />;
});
</code></pre>
<hr>
<h3>9. <strong>useLayoutEffect</strong></h3>
<p><strong>Kegunaan</strong>: Mirip useEffect, tetapi dijalankan secara sinkron setelah DOM dimutasi.<br>
<strong>Contoh</strong>:</p>
<pre><code class="language-jsx">useLayoutEffect(() => {
  // Ukur dimensi DOM di sini
  const width = element.current.offsetWidth;
}, [deps]);
</code></pre>
<hr>
<h3>10. <strong>useDebugValue</strong></h3>
<p><strong>Kegunaan</strong>: Menampilkan label kustom di React DevTools untuk custom hook.<br>
<strong>Contoh</strong>:</p>
<pre><code class="language-jsx">function useCustomHook() {
  const status = "Online";
  useDebugValue(status); // Tampilkan "Online" di DevTools
  return status;
}
</code></pre>
<hr>
<h3>11. <strong>useDeferredValue</strong> (React 18+)</h3>
<p><strong>Kegunaan</strong>: Menunda pembaruan UI untuk nilai tertentu (optimisasi performa).<br>
<strong>Contoh</strong>:</p>
<pre><code class="language-jsx">const deferredValue = useDeferredValue(value); // Tunda pembaruan &#x3C;code>value&#x3C;/code>
return &#x3C;List items={deferredValue} />;
</code></pre>
<hr>
<h3>12. <strong>useTransition</strong> (React 18+)</h3>
<p><strong>Kegunaan</strong>: Menandai pembaruan state sebagai transisi (non-blocking).<br>
<strong>Contoh</strong>:</p>
<pre><code class="language-jsx">const [isPending, startTransition] = useTransition();

startTransition(() => {
  setState(newState); // Pembaruan dianggap sebagai transisi
});
</code></pre>
<hr>
<h3>13. <strong>useId</strong> (React 18+)</h3>
<p><strong>Kegunaan</strong>: Membuat ID unik yang stabil di server dan client.<br>
<strong>Contoh</strong>:</p>
<pre><code class="language-jsx">const id = useId(); // Contoh output: ":r1:"
return &#x3C;label htmlFor={id}>Label&#x3C;/label>;
</code></pre>
<hr>
<h3>14. <strong>useSyncExternalStore</strong> (React 18+)</h3>
<p><strong>Kegunaan</strong>: Subscribe ke external store (e.g., state management eksternal).<br>
<strong>Contoh</strong>:</p>
<pre><code class="language-jsx">const state = useSyncExternalStore(
  store.subscribe,
  store.getState
);
</code></pre>
<hr>
<h3>Catatan:</h3>
<ul>
<li><strong>Hooks dengan use</strong> hanya bisa dipanggil di komponen fungsi atau custom hook.</li>
<li><strong>Dependency Array</strong> pada useEffect, useCallback, dan useMemo menentukan kapan efek/fungsi/nilai diperbarui.</li>
<li><strong>Hooks React 18+</strong> memerlukan versi React 18 atau lebih baru.</li>
</ul>
