f:T1c2e,<p>JavaScript menyediakan banyak metode bawaan pada objek Object yang berguna untuk memanipulasi, mengakses, dan memeriksa objek. Berikut adalah beberapa metode penting dari Object beserta penjelasannya:</p>
<hr>
<h3>1. <strong>Object.assign(target, ...sources)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Menyalin properti yang dapat di-enumerate (yang bisa di-loop) dari satu atau lebih objek sumber (sources) ke objek target (target).</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const target = { a: 1 };
const source = { b: 2, c: 3 };
const result = Object.assign(target, source);
console.log(result); // Output: { a: 1, b: 2, c: 3 }
</code></pre>
</li>
<li><strong>Catatan</strong>: Jika ada properti yang sama di target dan source, nilai dari source akan menimpa nilai di target.</li>
</ul>
<hr>
<h3>2. <strong>Object.create(proto, propertiesObject)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Membuat objek baru dengan prototipe yang ditentukan (proto) dan properti opsional (propertiesObject).</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const proto = { greet: function() { return "Hello!"; } };
const obj = Object.create(proto);
console.log(obj.greet()); // Output: "Hello!"
</code></pre>
</li>
</ul>
<hr>
<h3>3. <strong>Object.defineProperty(obj, prop, descriptor)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Menambahkan atau memodifikasi properti pada objek dengan deskripsi tertentu (seperti writable, enumerable, configurable, value, get, set).</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const obj = {};
Object.defineProperty(obj, 'name', {
  value: 'John',
  writable: false, // Tidak bisa diubah
  enumerable: true // Bisa di-loop
});
console.log(obj.name); // Output: "John"
</code></pre>
</li>
</ul>
<hr>
<h3>4. <strong>Object.defineProperties(obj, properties)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Menambahkan atau memodifikasi beberapa properti sekaligus pada objek.</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const obj = {};
Object.defineProperties(obj, {
  name: { value: 'John', enumerable: true },
  age: { value: 30, enumerable: true }
});
console.log(obj); // Output: { name: "John", age: 30 }
</code></pre>
</li>
</ul>
<hr>
<h3>5. <strong>Object.entries(obj)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Mengembalikan array dari pasangan [key, value] untuk setiap properti yang dapat di-enumerate pada objek.</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const obj = { a: 1, b: 2 };
console.log(Object.entries(obj)); // Output: [ ['a', 1], ['b', 2] ]
</code></pre>
</li>
</ul>
<hr>
<h3>6. <strong>Object.keys(obj)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Mengembalikan array dari semua properti yang dapat di-enumerate pada objek.</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const obj = { a: 1, b: 2 };
console.log(Object.keys(obj)); // Output: ['a', 'b']
</code></pre>
</li>
</ul>
<hr>
<h3>7. <strong>Object.values(obj)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Mengembalikan array dari semua nilai properti yang dapat di-enumerate pada objek.</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const obj = { a: 1, b: 2 };
console.log(Object.values(obj)); // Output: [1, 2]
</code></pre>
</li>
</ul>
<hr>
<h3>8. <strong>Object.freeze(obj)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Membekukan objek, sehingga propertinya tidak bisa diubah, ditambah, atau dihapus.</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const obj = { a: 1 };
Object.freeze(obj);
obj.a = 2; // Tidak akan berubah
console.log(obj.a); // Output: 1
</code></pre>
</li>
</ul>
<hr>
<h3>9. <strong>Object.seal(obj)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Menyegel objek, sehingga properti yang sudah ada tidak bisa dihapus, tetapi masih bisa diubah.</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const obj = { a: 1 };
Object.seal(obj);
obj.a = 2; // Bisa diubah
delete obj.a; // Tidak bisa dihapus
console.log(obj); // Output: { a: 2 }
</code></pre>
</li>
</ul>
<hr>
<h3>10. <strong>Object.is(value1, value2)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Membandingkan dua nilai secara strict (termasuk NaN dan +0/<code>-0</code>).</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">console.log(Object.is(NaN, NaN)); // Output: true
console.log(Object.is(0, -0)); // Output: false
</code></pre>
</li>
</ul>
<hr>
<h3>11. <strong>Object.getPrototypeOf(obj)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Mengembalikan prototipe dari objek.</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const proto = { greet: function() { return "Hello!"; } };
const obj = Object.create(proto);
console.log(Object.getPrototypeOf(obj) === proto); // Output: true
</code></pre>
</li>
</ul>
<hr>
<h3>12. <strong>Object.setPrototypeOf(obj, prototype)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Mengatur prototipe dari objek.</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const proto = { greet: function() { return "Hello!"; } };
const obj = {};
Object.setPrototypeOf(obj, proto);
console.log(obj.greet()); // Output: "Hello!"
</code></pre>
</li>
</ul>
<hr>
<h3>13. <strong>Object.hasOwnProperty(prop)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Memeriksa apakah objek memiliki properti tertentu (tidak termasuk properti dari prototipe).</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const obj = { a: 1 };
console.log(obj.hasOwnProperty('a')); // Output: true
console.log(obj.hasOwnProperty('toString')); // Output: false
</code></pre>
</li>
</ul>
<hr>
<h3>14. <strong>Object.isExtensible(obj)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Memeriksa apakah objek bisa ditambahkan properti baru.</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const obj = { a: 1 };
console.log(Object.isExtensible(obj)); // Output: true
Object.freeze(obj);
console.log(Object.isExtensible(obj)); // Output: false
</code></pre>
</li>
</ul>
<hr>
<h3>15. <strong>Object.preventExtensions(obj)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Mencegah penambahan properti baru ke objek.</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const obj = { a: 1 };
Object.preventExtensions(obj);
obj.b = 2; // Tidak akan ditambahkan
console.log(obj); // Output: { a: 1 }
</code></pre>
</li>
</ul>
<hr>
<h3>16. <strong>Object.fromEntries(iterable)</strong></h3>
<ul>
<li><strong>Fungsi</strong>: Mengubah array pasangan [key, value] menjadi objek.</li>
<li><strong>Contoh</strong>:
<pre><code class="language-javascript">const entries = [ ['a', 1], ['b', 2] ];
const obj = Object.fromEntries(entries);
console.log(obj); // Output: { a: 1, b: 2 }
</code></pre>
</li>
</ul>
<hr>
<h3>Kesimpulan</h3>
<p>Metode-metode di atas adalah alat yang sangat berguna untuk bekerja dengan objek di JavaScript. Setiap metode memiliki kegunaan spesifik, seperti manipulasi properti, pengaturan prototipe, atau pemeriksaan objek.
